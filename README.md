# Insider-Threat-Detection-for-AP-Transactions
Detect and prioritize insider-threat indicators in AP transactions using ACL rule-based testing, ML validation in Colab, and a management dashboard. using Power BI


Key Results (Final)
•	Illegal transactions: 0.44% | Legal transactions: 99.56%
•	Total illegal invoice amount: 84.69M
•	ACL Phase 2 outputs: Suspicious_Weekends = 5,250 | Conflict_Breaches = 1 | mismatch_legalty = 0
•	Best ML model: Random Forest (RF) | Confusion Matrix: [[3637, 0], [0, 16]]
